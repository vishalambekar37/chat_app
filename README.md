# Simple-Chat App

A simple Chat App for general purposes 

## Why use it?

It is useful for Chatting or Group Chat.

## Features
* Real-time Messaging
* User Authentication
* Offline Capabilities


## Built With
* Flutter
* Dart
* Firebase
